Introduction:

 This web app allows a user to create price alerts for stock tickers listed on the NASDAQ Global MarketSM exchange.

Users can access:
  • About - Includes website details along with the GitHub code repository.
  • API Documentation - Provides documentation on API access to interact with the app without a web browser.
  • Manage Stock Alerts - (Authenticated Users Only) Allows the user to create a stock price alert. The user will receive an email notification when the stock moves to the desired price.

    • Using 'Manage Stock Alerts':

     If the current price of TSLA (Tesla Inc) is $1067,
  • The user can select an alert for when the price goes up to a user-defined level such as $1100.
  • Once the real-time price of TSLA reaches or goes above $1100, the user will receive an email notification that the alert has been triggered.

    • Alerts can also be set for price drops (eg. if TSLA reaches or goes below a user defined value such as $1000).

    Misc Notes:
  • This website is intended for educational purposes only and should not be used for live trading.
  • Alerts are checked every 10 minutes. There is a high likelihood that a stock may cross across the desired alert price, then go back up/back down before the email notifier is triggerred.
  • There is a limit of 5 alerts per user.
