About

 This web app has been created to demonstrate understanding of back-end web development and structure, web architecture, HTTP requests and responses, RESTful API, SQL relational databases, and more.

Tools and tech stack used include:
  • Python 3.8 with Flask web framework
  • Docker containerization
  • Amazon AWS services (EC2, ECR, ECS, RDS using PostgreSQL)
  • CI/CD pipeline via GitHub Actions

    • Credits to the Flask Mega-Tutorial by Miguel Grinberg. Some components of this apps modules (such as login authentication and structuring the API framework) have been re-used from the tutorial with modifications as needed.
